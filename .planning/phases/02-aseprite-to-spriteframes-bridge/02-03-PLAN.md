<objective>
Implement sprite sheet splitting and atlas creation commands. `sprite split` divides an existing sprite sheet into frames (grid-based or JSON-defined) and generates a SpriteFrames .tres. `sprite create-atlas` composites multiple individual sprite images into a single atlas texture using shelf packing.

Purpose: These commands complete the sprite workflow beyond Aseprite imports. Users with non-Aseprite sprite sheets can still generate SpriteFrames, and users with individual frame images can create atlases for Godot.

Output: Two new library modules (`sprite/splitter.py`, `sprite/atlas.py`), two new CLI subcommands added to `commands/sprite.py`, and comprehensive tests.
</objective>

    **Functions:**

    1. `split_sheet_grid(image_path: Path, frame_w: int, frame_h: int, image_res_path: str) -> GdResource`:
       - Call `_require_pillow()`
       - Open image with `Image.open(image_path)` to get dimensions (width, height)
       - Validate: if frame_w > width or frame_h > height, raise ValidationError(code="SPRITE_FRAME_TOO_LARGE", fix=f"Frame size {frame_w}x{frame_h} exceeds image size {width}x{height}")
       - If width % frame_w != 0 or height % frame_h != 0: `warnings.warn(f"Image {width}x{height} not evenly divisible by {frame_w}x{frame_h}; partial edge frames ignored")`
       - Compute cols = width // frame_w, rows = height // frame_h
       - Create ExtResource for the sprite sheet texture (type="Texture2D", path=image_res_path)
       - Create AtlasTexture SubResources: for row in range(rows), for col in range(cols): region = Rect2(float(col * frame_w), float(row * frame_h), float(frame_w), float(frame_h))
       - Build one "default" animation containing all frames (same pattern as build_spriteframes default animation):
         ```python
         animation = {
             "frames": [{"duration": 1.0, "texture": SubResourceRef(sub.id)} for sub in sub_resources],
             "loop": True,
             "name": StringName("default"),
             "speed": 10.0,  # default FPS for grid split
         }
         ```
       - Compute load_steps = 1 + len(sub_resources) + 1
       - Return GdResource(type="SpriteFrames", format=3, uid=uid_to_text(generate_uid()), load_steps=load_steps, ext_resources=[ext], sub_resources=sub_resources, resource_properties={"animations": [animation]})
       - Close the Image after getting dimensions (just need width/height, no pixel manipulation)

    2. `split_sheet_json(image_path: Path, json_path: Path, image_res_path: str) -> GdResource`:
       - Call `_require_pillow()`
       - Read JSON metadata from json_path
       - JSON structure expected: `{"frames": [{"x": 0, "y": 0, "w": 32, "h": 32}, ...]}` or Aseprite-style regions
       - For each frame in JSON, create AtlasTexture SubResource with the specified region
       - Build "default" animation with all frames
       - Return GdResource

    **Functions:**

    1. `next_power_of_two(n: int) -> int`:
       - If n <= 0, return 1
       - If n is already power of two, return n
       - Return `1 << (n - 1).bit_length()`

    2. `create_atlas(image_paths: list[Path], atlas_res_path: str, power_of_two: bool = True) -> tuple[Any, GdResource]`:
       - Call `_require_pillow()`
       - If not image_paths: raise ValidationError(code="ATLAS_NO_IMAGES", fix="Provide at least one image file")
       - Open all images, record (width, height) for each
       - **Shelf packing algorithm (D-12):**
         - Sort images by height descending (tallest first for better packing)
         - Initialize: shelf_x=0, shelf_y=0, shelf_height=0, max_width=0
         - For each image:
           - If shelf_x + img.width > current atlas width estimate: move to next shelf (shelf_y += shelf_height, shelf_x=0, shelf_height=0)
           - Place image at (shelf_x, shelf_y)
           - Record placement: (path, x, y, w, h)
           - shelf_x += img.width
           - shelf_height = max(shelf_height, img.height)
           - max_width = max(max_width, shelf_x)
         - Final atlas dimensions: width=max_width, height=shelf_y+shelf_height
         - First pass: compute placements with a generous initial width estimate (sum of all widths, or sqrt of total area * 1.5)
       - If power_of_two: round width and height up to next power of two
       - Create atlas image: `Image.new("RGBA", (atlas_w, atlas_h), (0, 0, 0, 0))`
       - Paste each image at its placement position
       - Create GdResource:
         - ExtResource for atlas texture (type="Texture2D", path=atlas_res_path)
         - One AtlasTexture SubResource per placed image: region=Rect2(x, y, w, h)
         - One "default" animation with all frames, speed=10.0, loop=True
         - load_steps = 1 + len(sub_resources) + 1
       - Return (atlas_image, resource)
       - Close individual source images after pasting
