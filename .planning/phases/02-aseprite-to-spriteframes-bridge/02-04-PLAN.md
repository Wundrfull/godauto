    **Create `src/gdauto/sprite/validator.py`** (per D-16, SPRT-10):

    ```python
    from __future__ import annotations
    import tempfile
    from pathlib import Path
    from typing import Any
    from gdauto.errors import ValidationError, GodotBinaryError
    from gdauto.formats.tres import parse_tres_file, GdResource
    from gdauto.formats.values import SubResourceRef, ExtResourceRef, StringName
    ```

    **Functions:**

    1. `validate_spriteframes(path: Path) -> dict[str, Any]`:
       Structural validation (no Godot binary needed).

       Steps:
       a. Parse the .tres file: `resource = parse_tres_file(path)`. Catch ParseError and return {"valid": False, "issues": [str(error)]}.
       b. Check resource.type == "SpriteFrames". If not: issue "Resource type is '{resource.type}', expected 'SpriteFrames'".
       c. Check "animations" in resource.resource_properties. If not: issue "Missing 'animations' property in [resource] section".
       d. Get animations list from resource_properties["animations"]. If not a list: issue "animations property is not an array".
       e. Collect all sub_resource IDs: `sub_ids = {sub.id for sub in resource.sub_resources}`.
       f. Collect all ext_resource IDs: `ext_ids = {ext.id for ext in resource.ext_resources}`.
       g. For each animation dict in animations:
          - Check required keys: "name", "frames", "speed", "loop". Missing keys -> issue.
          - Check "name" is a StringName (or at least a string value).
          - Check "frames" is a list.
          - For each frame in "frames":
            - Check "texture" key exists and is a SubResourceRef.
            - If SubResourceRef, check .id is in sub_ids. Missing -> issue "Animation '{name}' frame {i} references unknown SubResource '{id}'".
            - Check "duration" key exists and is a number.
          - Check "speed" is a positive number. Zero or negative -> issue.
       h. For each sub_resource in resource.sub_resources:
          - If type == "AtlasTexture", check "atlas" property is ExtResourceRef.
          - If ExtResourceRef, check .id is in ext_ids. Missing -> issue.
          - Check "region" property exists and is a Rect2.
       i. Check load_steps: expected = len(ext_resources) + len(sub_resources) + 1. If mismatch -> issue (warning, not fatal).
       j. Build animation summaries: [{"name": anim_name, "frames": frame_count, "speed": speed, "loop": loop} for each animation].
       k. Return {"valid": len(issues) == 0, "animations": summaries, "issues": issues, "ext_resource_count": len(ext_resources), "sub_resource_count": len(sub_resources)}.

       Note on value type checking: After parse_tres_file, properties are Python objects (the parser converts them). SubResourceRef instances are already parsed. StringName instances are already parsed. Check with isinstance().

    2. `validate_spriteframes_headless(path: Path, backend: Any) -> dict[str, Any]`:
       Headless Godot validation.

       Steps:
       a. First run structural validation to catch obvious issues.
       b. Create a temporary GDScript that loads the resource and checks it:
          ```gdscript
          extends SceneTree

          func _init() -> void:
              var res = load("{res_path}")
              if res == null:
                  print("VALIDATION_FAIL: Could not load resource")
                  quit(1)
              if not res is SpriteFrames:
                  print("VALIDATION_FAIL: Resource is not SpriteFrames")
                  quit(1)
              var anims = res.get_animation_names()
              print("VALIDATION_OK: animations=" + str(anims.size()))
              for anim_name in anims:
                  var count = res.get_frame_count(anim_name)
                  print("ANIM: " + anim_name + " frames=" + str(count))
              quit(0)
          ```
       c. Write the script to a temp file.
       d. Run via backend.run(["--script", script_path], project_path=path.parent).
       e. Parse stdout for VALIDATION_OK/VALIDATION_FAIL.
       f. Return enriched result with Godot-confirmed data.
       g. Catch GodotBinaryError and fall back to structural-only result with a note.

    **Update `src/gdauto/commands/sprite.py`** to add the `validate` subcommand:

    ```python
    @sprite.command("validate")
    @click.argument("tres_file", type=click.Path(exists=False))
    @click.option("--godot", is_flag=True, default=False,
                  help="Also validate by loading in headless Godot (requires Godot binary).")
    @click.pass_context
    def validate(ctx: click.Context, tres_file: str, godot: bool) -> None:
        """Validate a SpriteFrames .tres resource file.

        Checks structure, animation definitions, frame references, and texture
        references. With --godot, also loads the resource in headless Godot
        to confirm it is valid.
        """
    ```

    Command body:
    1. Validate tres_file exists. If not, emit_error with FILE_NOT_FOUND.
    2. Call `validate_spriteframes(Path(tres_file))`.
    3. If --godot flag, also call `validate_spriteframes_headless()` with GodotBackend from ctx.obj.godot_path.
    4. Emit result:
       - JSON mode: output the full validation dict
       - Human mode: print summary ("Valid SpriteFrames with 3 animations (12 frames)") or list issues
    5. Exit code: 0 if valid, 1 if invalid.

    **Human output function:**
    ```python
    def _print_validate_result(data: dict[str, Any], verbose: bool = False) -> None:
        if data["valid"]:
            anims = data.get("animations", [])
            total_frames = sum(a.get("frames", 0) for a in anims)
            click.echo(f"Valid SpriteFrames with {len(anims)} animation(s) ({total_frames} frames)")
            if verbose:
                for anim in anims:
                    click.echo(f"  {anim['name']}: {anim['frames']} frames, {anim['speed']} FPS, loop={anim['loop']}")
        else:
            click.echo(f"Invalid SpriteFrames: {len(data['issues'])} issue(s)")
            for issue in data["issues"]:
                click.echo(f"  - {issue}")
    ```
