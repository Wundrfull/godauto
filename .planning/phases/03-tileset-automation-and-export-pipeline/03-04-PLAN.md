       ```python
       """Tiled .tmx/.tmj parser for basic tileset extraction (per D-08)."""
       from __future__ import annotations
       import json
       import xml.etree.ElementTree as ET
       from dataclasses import dataclass
       from pathlib import Path
       from gdauto.errors import ValidationError

       @dataclass
       class TiledTileset:
           """Minimal tileset data extracted from a Tiled file."""
           name: str
           tile_width: int
           tile_height: int
           columns: int
           tile_count: int
           image_path: str
           image_width: int
           image_height: int
           margin: int = 0
           spacing: int = 0

           @property
           def rows(self) -> int:
               """Compute row count from tile_count and columns."""
               if self.columns <= 0:
                   return 0
               return (self.tile_count + self.columns - 1) // self.columns
       ```

       Function parse_tiled_json(path: Path) -> list[TiledTileset]:
         - Read file, parse JSON
         - For each entry in data["tilesets"]:
           - Skip if "source" in entry and "tilewidth" not in entry (external .tsj reference)
           - Extract: name, tilewidth, tileheight, columns, tilecount, image, imagewidth, imageheight, margin, spacing
           - Validate required fields present (tilewidth, tileheight, columns, tilecount, image)
           - Create TiledTileset
         - Return list

       Function parse_tiled_xml(path: Path) -> list[TiledTileset]:
         - Parse XML with ET.parse(path)
         - Find all <tileset> elements
         - For each: extract attributes (name, tilewidth, tileheight, tilecount, columns)
         - Find <image> child element for source, width, height
         - Handle optional margin and spacing attributes (default 0)
         - Create TiledTileset
         - Return list

       Function parse_tiled_file(path: Path) -> list[TiledTileset]:
         - Dispatch based on suffix: .tmj/.json -> parse_tiled_json, .tmx/.xml -> parse_tiled_xml
         - Raise ValidationError if unsupported extension

       Function validate_tileset(path: Path) -> dict[str, Any]:
         - Parse with parse_tres_file, catch ParseError -> return invalid with parse error
         - Check resource.type == "TileSet"
         - Check resource_properties contains "tile_size"
         - Find all sub_resources where type == "TileSetAtlasSource"
         - If no atlas sources, add issue "No TileSetAtlasSource found"
         - For each atlas source:
           - Check "texture" key exists in properties (should be ExtResourceRef)
           - Check "texture_region_size" key exists
           - Count tiles: keys matching pattern r"^\d+:\d+/" (unique col:row prefixes)
           - Count terrain tiles: keys matching r"^\d+:\d+/terrain_set"
           - Count physics tiles: keys matching r"^\d+:\d+/physics_layer"
         - Check terrain_set consistency: if any tile references terrain_set=0, check that resource_properties has "terrain_set_0/mode" (pitfall 2 detection)
         - Build summary dict:
           ```python
           {
               "valid": len(issues) == 0,
               "type": resource.type,
               "tile_size": serialize_value(resource.resource_properties.get("tile_size", "unknown")),
               "atlas_sources": [
                   {
                       "tile_count": N,
                       "terrain_tiles": M,
                       "physics_tiles": P,
                       "has_texture": bool,
                   }
                   for each source
               ],
               "terrain_sets": count of terrain_set_N keys in resource_properties,
               "issues": issues,
               "warnings": warnings,
           }
           ```

       Function validate_tileset_headless(path: Path, backend: GodotBackend) -> dict[str, Any]:
         - First run validate_tileset for structural checks
         - If structural issues, return early
         - Create a temporary GDScript that loads the resource and checks:
           - Resource loads successfully
           - Type is TileSet
           - Has at least one source
           - Terrain sets are accessible
         - Run via backend.run() with the temp script
         - Parse output (print statements from GDScript) for pass/fail
         - Merge headless results into structural result
         - Same pattern as validate_spriteframes_headless
